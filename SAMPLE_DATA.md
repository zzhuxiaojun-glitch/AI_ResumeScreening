## How to Use These Samples

### Quick Test (5 minutes)

1. Create "Senior Frontend Developer" position
2. Upload `resume_excellent.txt` → Should get Grade A (80+)
3. Upload `resume_good.txt` → Should get Grade B (60-80)
4. Upload `resume_entry.txt` → Should get Grade C (40-60)
5. Upload `resume_reject.txt` → Should get low score + warning

### Full Test (10 minutes)

1. Create all 4 positions
2. Upload all 5 resumes to each position
3. Compare scoring differences across positions
4. Test filters, sorting, and export
5. Verify scoring logic is working correctly

### Custom Testing

Modify the resumes to test edge cases:
- Remove skills to see score drop
- Add reject keywords to see penalties
- Change education level
- Adjust work years
- Test with real PDF files (convert txt to PDF)
